Test Cases Reference

Sanitization Test Cases:

  • 123.45 → 123.45 (valid decimal)
  • abc123 → 123 (letters removed)
  • 12.345 → 12.34 (fraction truncated to 2 decimals)
  • 00123 → 123 (leading zeros removed)
  • 0.5 → 0.5 (leading zero kept for decimal)
  • 12.34.56 → 12.34 (multiple decimals handled)
  • $100.00 → 100.00 (currency symbol removed)
  • 1,000.50 → 1000.50 (comma removed)

Format Test Cases:

  • 1234.56 → 1,234.56 (thousands separator)
  • 100 → 100.00 (add decimal places)
  • 1234567.89 → 1,234,567.89 (multiple separators)
  • 0.5 → 0.50 (single decimal to two decimals)

Normalize Test Cases:

  • 123.456 → 123.46 (round to 2 decimals)
  • 123.4 → 123.40 (pad to 2 decimals)
  • 1,234.56 → 1234.56 (remove comma)
  • -100 → 0.00 (negative clamped to 0)

Max Button Behavior:

  • Max button sets value to balance.toFixed(2)
  • Max button disabled when balance ≤ 0

Preset Button Behavior:

  • Preset buttons set value to preset.toFixed(2)
  • Preset buttons disabled when preset > balance
  • Default presets: [100, 500, 1000]
